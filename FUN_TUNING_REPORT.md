# 符术派重构报告

## 概述
将符术派从"治疗辅助"重构为"抽牌连击流"，核心机制包括：抽牌、回灵气、降费、连锁出牌。

## 重构前状态
- 符术派以治疗为主（治愈符、强治愈符、还原符、突破符）
- 缺少能量机制
- 缺乏连击/连锁系统
- 卡牌同质化严重

## 新增卡牌机制

### 类型: skill
新增卡牌类型 `skill`（技能），用于抽牌、降费、回能量等非攻击/防御/治疗的卡牌。

### 卡牌属性
- `drawCards`: 抽牌数量
- `gainEnergy`: 获得灵气
- `reduceCost`: 费用减少（对下一张牌生效）
- `chainDraw`: 连锁抽牌（本回合每出一张牌则抽）

### 玩家状态
```typescript
interface PlayerState {
  // ... existing
  pendingCostReduction?: number; // 本回合下一张牌费用减少
  cardsPlayedThisTurn?: number; // 本回合已出牌数
}
```

## 连锁Combo系统

### 触发条件
1. 出牌时递增 `cardsPlayedThisTurn`
2. 无尽符效果：每出1张牌，抽 `chainDraw × cardsPlayedThisTurn` 张

### 示例combo
1. 灵引(0费抽1) → 出牌数=1
2. 双引(1费抽2) → 出牌数=2
3. 无尽符(3费) → 本回合已出3张，触发连锁抽3张

### UI提示
- 降费状态显示 "费用-1" 标签
- 连锁状态显示 "x2"、"x3" 等combo计数
