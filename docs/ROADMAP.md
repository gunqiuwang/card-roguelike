**目标**：核心钩子 #1 —— 收服妖怪。

- [x] 封印成功：妖 → 卡入组（战后进弃牌堆并整合）
- [x] 4 只可封妖（青丘章节）+ 12 张卡池（starter 4 + common 2 + rare 4 + epic 2 + legend 2）
- [x] 敌 AI 意图：攻击 / 防御 / 蓄力 / 施法 + 蓄力大招
- [x] 战后回血、灵气奖励、3 选 1 卡牌奖励（稀有度加权）
- [x] 祭坛：升卡 / 删卡 / 静修 / 离去
- [x] 奇遇：3 个事件（荒坟 / 古井 / 符贩），每个 3 选 1

- [x] UI：TitleScreen / MapScreen / BattleScreen / RewardScreen /

## v0.3 · 拼符系统 ✅ 已上线

**目标**：核心钩子 #2 —— 封印有操作感。

- [x] 封印小游戏 UI（笔画点击）
- [x] 6 种笔画元素 + 10 种符阵
- [x] 完美/成功/失败的差异化奖励
- [x] 随章节解锁更多笔画

**v0.3 已实现内容**：
- 每笔画倒计时（普通 3s，Boss/精英 1.5s）
- 10 种符阵 SVG 背景，笔画渐进绘制动画
- 笔画正确/错误视觉反馈（闪绿/抖红）+ 提示笔画金色脉冲
- 完美封印（限时全对无失误）→ 升阶妖卡 +3 伤害
- 失败：扣 8HP，妖恢复 50% HP，可重新触发封印

## v0.4 · 妖性系统 + 图鉴 ✅ 已上线

**目标**：长线收集欲 + 使用代价。

- [x] 妖卡的妖性值（0-100）
- [x] 反噬事件（噩梦场景）
- [x] 祭坛"驯妖"
- [x] 图鉴页面（封过的妖档案）
- [x] 解锁式 meta 进度

**v0.4 已实现内容**：
- 妖卡打出时妖性 +5（定神卷减半）
- 妖性 ≥ 60 躁动（10% 概率自残 3 HP）
- 妖性 ≥ 90 狂乱（必自残 5 HP + 25% 丢弃手牌）
- 战斗结束后所有妖卡妖性 -1（闲置衰减）
- 祭坛「驯妖」消耗 25 灵气，降低 20 妖性
- 夜间反噬：平均妖性 ≥ 50 时，非战斗节点后扣 10% maxHP
- 祭坛/奇遇离开时重置反噬标记，支持多次触发
- 图鉴显示已封印妖的档案（含封后卡信息）

## v0.5 · 阴阳双道职业 ✅ 已上线

**目标**：高玩流派，玩法深度天花板。

- [x] 阴阳师职业（双能量池，左阴右阳）
- [x] 阴阳平衡机制（偏阴≥3 +1力量，偏阳≥3 +1易伤）
- [x] "太极归一"终极技能（阴阳归零时可用，30伤+10治疗）
- [x] 阴卡/阳卡区分，能量池隔离
- [x] 阴出阳增，阳出阴增（平衡累积）

**v0.5 已实现内容**：
- 阴阳双道职业：阴/阳两套独立能量池（各3/回合）
- 偏阴平衡触发力量 +1，偏阳平衡触发易伤 +1
- 太极归一：消耗所有阴阳积蓄，对全体敌人造成 30 点伤害，回复 10 气血
- 阴阳能量池独立：阴卡消耗阴，阳卡消耗阳
- 平衡更新：出阴卡则阳平衡 +1，出阳卡则阴平衡 +1
