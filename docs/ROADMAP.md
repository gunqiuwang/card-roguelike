## v0.2 · 封妖 MVP（本 PR · 已实装）

**目标**：实现核心钩子 #1 —— 收服妖怪。

- [x] 回合制战斗系统（抽牌/出牌/弃牌/能量/状态）
- [x] 节点地图（线性 8 节点：2 普通 / 1 精英 / 1 Boss / 1 奇遇 / 1 祭坛 / 2 普通·混编）
- [x] 敌人 HP ≤ 30% 时弹出【斩/封】选择
- [x] 封印成功：妖 → 卡入组（战后进弃牌堆并整合）
- [x] 4 只可封妖（青丘章节）+ 12 张卡池（starter 4 + common 2 + rare 4 + epic 2 + legend 2）
- [x] 敌 AI 意图：攻击 / 防御 / 蓄力 / 施法 + 蓄力大招
- [x] 战后回血、灵气奖励、3 选 1 卡牌奖励（稀有度加权）
- [x] 祭坛：升卡 / 删卡 / 静修 / 离去
- [x] 奇遇：3 个事件（荒坟 / 古井 / 符贩），每个 3 选 1
- [x] 牌组上限 20 的溢出处理
- [x] 图鉴（封过的妖解锁档案）
- [x] 仿真器 v1：跑 1000 局胜率验收（首版 43-47%）
- [x] 存档（localStorage，单槽 + schema 版本锁）
- [x] UI：TitleScreen / MapScreen / BattleScreen / RewardScreen /
      EventScreen / ShrineScreen / OverflowScreen / VictoryScreen /
      GameOverScreen / CodexScreen
- [x] 引擎 smoke 测试（12 条 path 全绿）

**PR**：feat/v0.2-full-game · 一个 PR 全量落地
