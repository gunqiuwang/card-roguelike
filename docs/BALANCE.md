## §5 · 经济系统

### §5.1 灵气奖励（斩 vs 封）

| 节点 | 斩：灵气 | 斩：卡牌奖励 | 封：灵气 | 封：卡牌奖励 |
|---|---|---|---|---|
| 普通战 C | 10~15 | 3 选 1 凡/珍 | 2~4 | 无（得妖卡） |
| 精英战 B | 25~35 | 3 选 1 凡/珍/灵 | 6~9 | 无 |
| Boss 战 S | 50~60 | 3 选 1 珍/灵/玄 | 12~15 | 无 |

**封妖经济比（vs 斩）**：约 **20%**。这是故意压低的——要让斩始终有吸引力，封才有"沉没成本"的张力。

### §5.2 祭坛消耗

| 动作 | 花费 |
|---|---|
| 升级一张卡（+伤/+御） | 30 灵气 |
| 删除一张卡 | 20 灵气 |
| 驯妖（妖性 -10）（v0.4） | 25 灵气 |
| 买秘卷（被动 buff）（v0.3） | 40 灵气 |

### §5.3 第一章预计资源流

```
起手: 0 灵气
战 1 青狐 x1 (斩): +12     → 12
战 2 青狐 x2 (斩/斩): +24   → 36
奇遇 (A 斩): +10            → 46
战 3 青+夜 (封青 / 斩夜): +3 +15 → 64
祭坛: 买秘卷(40)            → 24
精英 (封): +7               → 31
战 4 夜狐 x2 (斩/斩): +30   → 61
Boss (斩): +55              → 116 余裕
```

**目标**：玩家在 Boss 前应该**攒够 1 次祭坛消费 + 1 件秘卷 + 1 次卡升级**，也就是 ~90 灵气。上述流水显示合理。

## §6 · 战后回血

| 胜利类型 | 回血比例 |
|---|---|
| 普通战 | maxHp × 10% = 7 HP |
| 精英战 | maxHp × 15% = 10 HP |
| Boss 战 | maxHp × 30% = 21 HP + 永久 maxHp +5 |
| 封妖胜利 | 回血 × 0.5（承担因果代价） |

**第一章模拟玩家 HP 曲线**（假设每战损 8）：

| 节点 | HP 损 | HP 回 | HP 净 |
|---|---|---|---|
| 70 起手 | | | 70 |
| 战 1 | -8 | +7 | 69 |
| 战 2 | -10 | +7 | 66 |
| 奇遇 | 0 | 0 | 66 |
| 战 3 | -10 | +7 | 63 |
| 精英 (封) | -15 | +10×0.5=5 | 53 |
| 祭坛 | 0 | 0 | 53 |
| 战 4 | -10 | +7 | 50 |
| Boss | -30 | +21 | 41 |

**结论**：进入第二章时 HP 约 41/75（maxHp +5），**压力感合适**。

## §7 · 状态 / Status 系统

| StatusKind | 触发时机 | 效果 | 衰减 |
|---|---|---|---|
| `poison` | 每回合开始 | 持该角色 -1 HP / 层 | 每回合 -1 层（直到 0） |
| `weak` | 持续 N 回合 | 其攻击伤害 × 0.75 | 每回合 -1 层 |
| `vulnerable` | 持续 N 回合 | 其受到伤害 × 1.25 | 每回合 -1 层 |
| `strength` | 永久（Boss） | 其攻击伤害 +1 / 层 | 无 |
| `intentSealed` | 1 回合 | 下回合意图空过 | 自动消失 |
