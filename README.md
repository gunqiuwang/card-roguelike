# Card Roguelike

卡牌Roguelike游戏，类似Slay the Spire的轻量玩法。

## 技术栈

- React 18 + Vite + TypeScript
- Tailwind CSS 4.0
- Zustand (状态管理)

## 本地运行

```bash
npm install
npm run dev
```

## 构建

```bash
npm run build
```

## 部署

已配置 Vercel 自动部署，推送到 main 分支自动部署。

- GitHub: https://github.com/gunqiuwang/card-roguelike
- Vercel: https://card-roguelike-six.vercel.app/
